# HG6145F1 Password Generator (AT Nightmares)

Android GUI application for generating **admin** and **Wi-Fi passwords**
for **FiberHome HG6145F1 routers (Algérie Télécom)** using the router
MAC address.

This tool is designed for recovery and educational use.

## Features

-   Admin password generation\
-   Wi-Fi password generation\
-   Simple and fast Android GUI\
-   Offline operation\
-   No Python or PC required

## Requirements

-   Android phone\
-   APK installation enabled\
-   Disable **Auto Blocker / Play Protect** before installation (app is
    not from Play Store)

## How to Use

1.  Install the APK on your phone\
2.  Disable **Auto Blocker / Play Protect** if prompted\
3.  Open the app\
4.  Tap the **Gen** button\
5.  Paste the router **MAC address** when requested\
6.  The app instantly generates:
    -   Admin password\
    -   Wi-Fi password

## Supported Device

-   FiberHome **HG6145F1**\
-   Algérie Télécom firmware variants
